# Dotfiles

## Getting Started

#### 1. Create a code directory

```
$ mkdir ~/code
```

#### 2. Clone this repo:

```
$ git clone git@github.com:patwey/dotfiles.git ~/code/dotfiles
```

#### 2. Change to the dotfiles directory:

```
$ cd ~/code/dotfiles
```

#### 3. Run the setup script

```
$ sh setup.exclude.sh
```
